# Cosplay 图片搜索

本目录负责借助图片搜索库为每个角色拉取 cosplay 候选图。

## 需求说明

使用 **better-bing-image-downloader** 对每个角色搜索 cosplay 图，取 **top 5** 结果。

流程要点：

- 对每个角色调用 **gpt-5-mini**（或项目当前使用的等价 LLM）获取**简化角色名**
- 搜索词形式：`{简化角色名} cosplay {简化番剧名}`
- 尽量成功获取前 5 张图，保存到 `local_data`

实验与批量：

- 先在 **top 30** 角色上实验结果
- 另写脚本支持在服务器上对 **top 1000** 批量下载

展示：

- 提供 **HTML** 页面展示 top 30 的搜索结果（**原角色图 + 5 张结果图** 对比）
